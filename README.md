cgi-run
=======

Runner lib for cgi exec


## Configuration

### `options.root`

Document root for cgi script

### `options.script`

Default script to be executed, relative to `options.root`

### `options.cgi`

CGI bin path; defaults to `php-cgi`

### `options.extension`

Script extention name; defaults to `.php`

### `options.stderr`

`STDERR` for cgi bin
